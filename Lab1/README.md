Lab1-Linker
===========

Linker lab for Professor Hubertus Franke's OS course

Compile:
-----------
$ cd build  
$ make

Usage:
-----------
$ ./linker [OPTION]... FILE ...  
Options:	-c	Conserve memory by reading input files twice  
		-f	Enable faster processing by keeping modules in memory (default)  
		-h	Print this help message
